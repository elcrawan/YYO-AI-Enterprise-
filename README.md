### ๐ Installation & Setup

#### Prerequisites
- Node.js 18.0.0 or higher
- npm 8.0.0 or higher
- PostgreSQL 13 or higher
- Redis 6 or higher
- MongoDB 5 or higher (optional)

#### Installation Steps

1. **Clone the repository**
```bash
git clone https://github.com/elcrawan/YYO-AI-Enterprise-.git
cd YYO-AI-Enterprise-
```

2. **Install dependencies**
```bash
npm install
```

3. **Setup environment variables**
```bash
cp .env.example .env.local
```

4. **Configure environment file**
```env
# Databases
DATABASE_URL="postgresql://username:password@localhost:5432/yyo_ai_enterprise"
MONGODB_URI="mongodb://localhost:27017/yyo_ai_enterprise"
REDIS_URL="redis://localhost:6379"

# Authentication
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET="your-secret-key-here"

# AI Services
OPENAI_API_KEY="your-openai-api-key"
GOOGLE_AI_API_KEY="your-google-ai-api-key"
ANTHROPIC_API_KEY="your-anthropic-api-key"
# ... other keys
```

5. **Setup database**
```bash
npx prisma generate
npx prisma db push
```

6. **Run the system**
```bash
npm run dev
```

7. **Open browser**
```
http://localhost:3000
```
